== Installation ==

<p>nstallation of <strong>Ultimate Team Member</strong> is very simple. Here is two ways you can install plugin in your wordpress site:</p>

<strong>Option 1</strong>

<li> Download the plugin ultimate-team-member.zip from download location</li>
<li> Unpack the *.zip file and extract the Ultimate Team Member</li>
<li> Drop the Ultimate Team Member plugin folder into your wp-content/plugins folder.</li>
<li> In word press administration panels, click on Plugins from the menu.</li>
<li> You should see your plug-in listed under Inactive plug-in tab.</li>
<li> To turn the word presses plug-in on, click activate.</li>

<strong>Option 2</strong>

<li> Download the plugin Ultimate Team Member Plugin from download location.</li>
<li> Go to Add New menu under Plugins tab in your word-press admin.</li>
<li> Select upload link (top link menu).</li>
<li> Upload the available Ultimate Team Member Plugin file and click install now</li>
<li> Finally click activate plug-in link to activate the plug-in.</li>


<h3>Add Team Member</h3>	
<p>After Activate <strong>Ultimate Team Member</strong> Plugin. Go to WordPress Dashboard. In Dashboard Menu, you will see <strong>Team Member</strong> Custom Post Type which help you to Add Team Members.</p>
<p>Click add new, <strong>Ultimate Team Member</strong> option page will Open with Customize Options. In Option Page: You Should First Select Team Style: choose style then Click Add New Team Member.</p>
<p>After Clicking <strong> Add New Team Member</strong> Member Information Option Will Open. Fill all the information.</p>
<p>Then Click Again Add New Team Member at the last of information form</p>
<p>You Can also copy a Team member Items if you created before.</p>
<p>This options are used for changing Team Options for you team member.</p>

<h3>Team Options:</h3>	
<p><strong>Items in Column:</strong> This option are used how many items show in 1 row. If you create 3 team member then you should choose <strong>3 items</strong</p>
<p><strong>Items Center?:</strong> This option use if you want your Team member items align centered.</p>
<p><strong>Icon link new tab? :</strong> This option use if you want Social Icon Link Open in new Tab.</p>

<h3>Typography Font:</h3>	
<p><strong>Font Family:</strong> Choose font family from dropdown. there are 500+ google font you can use. and you can use also font weight for your font family.</p>
<p><strong>Name Font Size:</strong> You can change Member Name font size here. Default is 24px.</p>
<p><strong>Label Font Size:</strong> You can change Member Label font size. Default is 15px.</p>
<p><strong>Description Font Size:</strong> This option is use for change description font size. default is 12px.</p>

<h3>Typhography Color:</h3>	
<p><strong>Team Main Color:</strong> This is for your team member style main color.</p>
<p><strong>Member Info Color:</strong> There is 3 option for member information color. you can change color as your requirement.</p>
<p><strong>Social Icon Area:</strong> There is 3 option for social icon area. you can change color as your requirement for your social icon.</p> 

<h3>Publish:</h3>	
<p>When Ultinate Team Member items Created, just Click in Publish Button, You will see a shortcode which is created avobe.  Now we have to add shortCode on post or page where you want to show.</p>

<p> EX Shortcode Below:</p>

<pre>[[utm id='57']]</pre>
